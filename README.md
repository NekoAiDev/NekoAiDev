## About Us

Neko Ai is an open-source development organization dedicated to building plugins for the **AstrBot** framework. Our mission is to make AstrBot more powerful, more flexible, and more fun.

We believe that good tools should be intuitive, reliable, and extensible. Every plugin we build follows these principles:

- **User-Friendly** — Clean configs, clear feedback, easy to set up
- **Reliable** — Thoroughly tested, gracefully handling edge cases
- **Open Source** — MIT licensed, contributions welcome

Our plugins are published on the [AstrBot Plugin Marketplace](https://plugins.astrbot.app), making them easy to discover and install for all AstrBot users.

## Projects

### [astrbot_plugin_api_limiter](https://github.com/NekoAiDev/astrbot_plugin_api_limiter)

<img src="https://img.shields.io/github/v/release/NekoAiDev/astrbot_plugin_api_limiter?style=flat-square" alt="Release"/> <img src="https://img.shields.io/badge/License-MIT-green?style=flat-square" alt="License"/> <a href="https://github.com/NekoAiDev/astrbot_plugin_api_limiter/stargazers"><img src="https://img.shields.io/github/stars/NekoAiDev/astrbot_plugin_api_limiter?style=flat-square" alt="Stars"/></a> <a href="https://github.com/NekoAiDev/astrbot_plugin_api_limiter/forks"><img src="https://img.shields.io/github/forks/NekoAiDev/astrbot_plugin_api_limiter?style=flat-square" alt="Forks"/></a>

**API Rate Limiter** — A comprehensive rate limiting plugin for AstrBot that helps you manage API call frequency and prevent abuse.

**Features:**
- Interval-based call limiting — control the minimum time between API calls
- Count-based limiting with cooldown reset — set daily/hourly call quotas that reset automatically
- Quiet hours — automatically disable API calls during specified time periods
- Whitelist & blacklist — fine-grained control over which users/groups are affected
- Time-based rate limiting — different limits for different times of day
- Over-limit reminders — notify users when they're approaching or exceeding their limits
- Per-group independent quotas — each group chat can have its own rate limit config
- Usage logging & export — track and export API usage data
- Admin reset commands — easily reset counters when needed
